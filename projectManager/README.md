APIs:

* /list/
    * Provides all the lists of tasks

* /list/&lt;id&gt;/
    * provides list with id proovided in URL

* /list/&lt;id&gt;/task/
    * provides all the tasks in list with id in URL

* /task/
    * provides list of all the tasks

* /task/&lt;id&gt;/
    * provides the information of task with id in URL.
